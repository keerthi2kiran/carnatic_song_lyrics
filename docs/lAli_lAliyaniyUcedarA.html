ಲಾಲಿ ಲಾಲಿಯನಿಯೂಚೆದರಾ (lAli lAliyaniyUcedarA )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಹರಿ ಕಾಂಭೋಜಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಲಾಲಿ ಲಾಲಿಯನಿಯೂಚೆದರಾ ವನ
ಮಾಲಿ ಮಾಲಿಮಿತೋ ಜೂಚೆದರಾ

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
1. ದೇವ ದೇವಯನಿ ಪಿಲಿಚೆದರಾ ಮಹಾನು-
ಭಾವ ಭಾವಮುನ ತಲಚೆದರಾ (ಲಾಲಿ)

2. ರಾಮ ರಾಮಯನಿ ಪಲಿಕೆದರಾ ನಿಷ್-
ಕಾಮ ಕಾಮ ರಿಪು ನುತ ರಾರಾ (ಲಾಲಿ)

3. ಕೋರಿ ಕೋರಿ ನಿನ್ನು ಕೊಲಿಚೆದರಾ ಮಾಯ
ದಾರಿ ದಾರಿ ಸದ್-ಭಜನರಾ (ಲಾಲಿ)

4. ರಾಜ ರಾಜಯನಿ ಪೊಗಡೆದರಾ ತ್ಯಾಗ-
ರಾಜ ರಾಜಯನಿ ಮ್ರೊಕ್ಕೆದರಾ (ಲಾಲಿ)
