ರೇ ಮಾನಸ ಚಿಂತಯ (rE mAnasa cintaya )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ತೋಡಿ
ತಾಳ: ರೂಪಕ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ರೇ ಮಾನಸ ಚಿಂತಯ ಶ್ರೀ ರಾಮಂ

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
1. ಸಾಕೇತ ನಗರ ನಾಥುನಿ
ಸಮಾನ ರಹಿತ ಮಾ ವರುನಿ (ರೇ)

2. ಗತ ಕಾಮ ಜನ ಹೃದಯಾರ್ಣವ
ಕಲಾ ಧರುನಿ ಇಳಾಧಿಪುನಿ (ರೇ)

3. ಭಕ್ತಾಘ ಪಯೋ-ಧರ
ಪ್ರಭಂಜನುನಿ ನಿರಂಜನುನಿ (ರೇ)

4. ವಿನತಾ ತನಯಾರ್ತಿ ತಮೋ-
ವಿಭಾ-ಕರುನಿ ಶುಭಾ-ಕರುನಿ (ರೇ)


5. ದೇವೇಶ ಸಮಾರಾಧ್ಯುನಿ
ದಿಗಂಬರುನಿ ಚಿದಂಬರುನಿ (ರೇ)


6. ವಂದಾರು ಬೃಂದಾರಕ
ವರಾಭಯದುನಿ ರಾಘವುನಿ (ರೇ)

7. ವರ ತ್ಯಾಗರಾಜಾರ್ಚಿತ
ಪದಾಂಬುಜುನಿ ಸದಾ ಹಿತುನಿ (ರೇ)
