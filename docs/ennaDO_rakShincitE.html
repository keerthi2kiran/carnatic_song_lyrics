ಎನ್ನಡೋ ರಕ್ಷಿಂಚಿತೇ (ennaDO rakShincitE )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಸೌರಾಷ್ಟ್ರ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಎನ್ನಡೋ ರಕ್ಷಿಂಚಿತೇ ನೀವಿಪುಡೀ
ವಗನು ಉಂಡ ವಲೆನಾ

ಅನುಪಲ್ಲವಿ:
ಪನ್ನಗ ಶಯನ ಪೆದ್ದಲ ಸ್ವ-
ಭಾವಮಿಟ್ಲುಂಡಗ ಬಾಗುಗ ನನ್ನೇ ನೀವ್(ಎನ್ನಡೋ)

ಚರಣ:
1. ಚೀಕಟಿ ಚೇತನು ಗಾಸಿ ಜೆಂದಿಯೀ
ಲೋಕುಲು ಸಕಲ ಕರ್ಮಮುಲಕೈ ಮನ
ರಾಕ ಕೊರಕು ಜೂಚೆದರನಿ ತೆಲಿಸಿ
ವಿಭಾಕರಡು ಮದಿನಿ
ಪ್ರಾಕಟಮುಗ ತಾ ರಾನು ಚೆಲ್ಲುನನಿ
ವ್ಯಾಕುಲುಡೈ ಅತಿ ವೇಗಮುನನು
ವೇಕುವ ಜಾಮುನ ತೋಷಮು ಕಲಗನು
ತಾ ಕರುಣನರುಣುನಿ ಪನುಪ ಲೇದಾ (ಎನ್ನಡೋ)

2. ನೀರು ಲೇಕ ಸಜ್ಜನ ಗಣಮುಲು ಮನ
ದಾರಿನಿ ಜೂಚೇರನುಚು ತೆಲಿಸಿ
ಕಾವೇರಿ ತಾನು ರಾನು ಚೆಲ್ಲುನನುಚು
ವಿಚಾರ ಹೃದಯುರಾಲೈ
ಸಾರೆಕು ವಚ್ಚುನನುಚು ಸಂತೋಷಮು
ವೀರುಲಕೆಲ್ಲನು ಕಲ್ಗ ಪಡಮಟಿ
ಮಾರುತ ಚಂಚಲಮುಲ ಪನಿಪಿ
ಚೆಲಮಲೂರ ಚೇಯ ಲೇದಾ ರಾಮ (ಎನ್ನಡೋ)

3. ಗರ್ವಿಂಚಿನ ಆ ರಾವಣ ಬಾಧಲನ್-
ಓರ್ವ ಲೇಕನು ವಿಭೀಷಣುಡು ನಿನ್ನು
ಸರ್ವ ಲೋಕ ಶರಣ್ಯಾ ವಂದನಮನ
ನಿರ್ವಿಕಾರುಡೈ ನೀವು
ಸರ್ವದಾ ರಕ್ಷಿಂಚೆದನನುಚುನು
ಸಾರ್ವಭೌಮ ಲಂಕಾಧಿಪತ್ಯಮುನ
ಸರ್ವಜ್ನುನಿ ಲಕ್ಷ್ಮಣುನಿಮ್ಮನಗಾ
ಸರ್ವ ಸನ್ನುತ ತ್ಯಾಗರಾಜುನಿ (ಎನ್ನಡೋ)
