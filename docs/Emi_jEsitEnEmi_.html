Emi jEsitEnEmi

English Lyrics

Composer: tyAgarAja
Raga: thodi
Tala: miSra chApu
Language: telugu

Pallavi:
Emi jEsitEnEmi shrI rAma
swAmi karuNa lEni varilalO

Anupallavi:
kAma mOha dAsulai shrI rAmuni
kaTTu teliya lEni vArilalO (nEmi)

Charanam:
1. savamu jEsitEnEmi kalimini
putrOtsavamu kaligitEnEmi
bhuvilOnanya bIja janituni koniyEmi
shiva kara shrI rAmuni daya leni vArilalOn(Emi)


2. mEDa kaTTitEnEmi anduna lAndaru
jODu kaTTitEnEmi
cEDiyalanu meppinca telisitEnEmi
IDu lEni rAmuni daya lEni vArilalOn(Emi)

3. immu kaligitEnEmi illAliki
sommu peTTitEnEmi
kamma viltu kELini telisiyEmi
tammi kaNTi vAni karuNa lEni vArilalOn(Emi)


4. rAjyamElitEnEmi bahu janulalO
pUjyulaitEnEmi
Ajya pravAhamutOnannamiDitEnEmi
pUjyuDaina rAmuni daya lEni vArilalOn(Emi)


5. guruvu tAnaitEnEmi kaNTiki mEnu
guruvai tOcitEnEmi
vara mantramanyulakupadEsincitEnEmi
vara tyAgarAja nutuni daya lEni vArilalOn(Emi)

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ತೋಡಿ
ತಾಳ: ಮಿಶ್ರ ಛಾಪು
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಏಮಿ ಜೇಸಿತೇನೇಮಿ ಶ್ರೀ ರಾಮ
ಸ್ವಾಮಿ ಕರುಣ ಲೇನಿ ವರಿಲಲೋ

ಅನುಪಲ್ಲವಿ:
ಕಾಮ ಮೋಹ ದಾಸುಲೈ ಶ್ರೀ ರಾಮುನಿ
ಕಟ್ಟು ತೆಲಿಯ ಲೇನಿ ವಾರಿಲಲೋ (ನೇಮಿ)

ಚರಣ:
1. ಸವಮು ಜೇಸಿತೇನೇಮಿ ಕಲಿಮಿನಿ
ಪುತ್ರೋತ್ಸವಮು ಕಲಿಗಿತೇನೇಮಿ
ಭುವಿಲೋನನ್ಯ ಬೀಜ ಜನಿತುನಿ ಕೊನಿಯೇಮಿ
ಶಿವ ಕರ ಶ್ರೀ ರಾಮುನಿ ದಯ ಲೆನಿ ವಾರಿಲಲೋನ್(ಏಮಿ)


2. ಮೇಡ ಕಟ್ಟಿತೇನೇಮಿ ಅಂದುನ ಲಾಂದರು
ಜೋಡು ಕಟ್ಟಿತೇನೇಮಿ
ಚೇಡಿಯಲನು ಮೆಪ್ಪಿಂಚ ತೆಲಿಸಿತೇನೇಮಿ
ಈಡು ಲೇನಿ ರಾಮುನಿ ದಯ ಲೇನಿ ವಾರಿಲಲೋನ್(ಏಮಿ)

3. ಇಮ್ಮು ಕಲಿಗಿತೇನೇಮಿ ಇಲ್ಲಾಲಿಕಿ
ಸೊಮ್ಮು ಪೆಟ್ಟಿತೇನೇಮಿ
ಕಮ್ಮ ವಿಲ್ತು ಕೇಳಿನಿ ತೆಲಿಸಿಯೇಮಿ
ತಮ್ಮಿ ಕಂಟಿ ವಾನಿ ಕರುಣ ಲೇನಿ ವಾರಿಲಲೋನ್(ಏಮಿ)


4. ರಾಜ್ಯಮೇಲಿತೇನೇಮಿ ಬಹು ಜನುಲಲೋ
ಪೂಜ್ಯುಲೈತೇನೇಮಿ
ಆಜ್ಯ ಪ್ರವಾಹಮುತೋನನ್ನಮಿಡಿತೇನೇಮಿ
ಪೂಜ್ಯುಡೈನ ರಾಮುನಿ ದಯ ಲೇನಿ ವಾರಿಲಲೋನ್(ಏಮಿ)


5. ಗುರುವು ತಾನೈತೇನೇಮಿ ಕಂಟಿಕಿ ಮೇನು
ಗುರುವೈ ತೋಚಿತೇನೇಮಿ
ವರ ಮಂತ್ರಮನ್ಯುಲಕುಪದೇಸಿಂಚಿತೇನೇಮಿ
ವರ ತ್ಯಾಗರಾಜ ನುತುನಿ ದಯ ಲೇನಿ ವಾರಿಲಲೋನ್(ಏಮಿ)
