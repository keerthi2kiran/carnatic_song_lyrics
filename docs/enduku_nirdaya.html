ಎಂದುಕು ನಿರ್ದಯ (enduku nirdaya )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಹರಿ ಕಾಂಭೋಜಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಎಂದುಕು ನಿರ್ದಯ ಎವರುನ್ನಾರುರಾ

ಅನುಪಲ್ಲವಿ:
ಇಂದು ನಿಭಾನನ ಇನ ಕುಲ ಚಂದನ (ಎಂದುಕು)

ಚರಣ:
1. ಪರಮ ಪಾವನ ಪರಿಮಳಾಪ ಘನ (ಎಂದುಕು)

2. ನೇ ಪರ ದೇಶಿ ಬಾಪವೇ ಗಾಸಿ (ಎಂದುಕು)

3. ಉಡತ ಭಕ್ತಿ ಕನಿ ಉಬ್ಬತಿಲ್ಲಗ ಲೇದಾ (ಎಂದುಕು)

4. ಶತ್ರುಲ ಮಿತ್ರುಲ ಸಮಮುಗ ಜೂಚೇ ನೀಕ್(ಎಂದುಕು)

5. ಧರಲೋ ನೀವೈ ತ್ಯಾಗರಾಜುಪೈ (ಎಂದುಕು)
