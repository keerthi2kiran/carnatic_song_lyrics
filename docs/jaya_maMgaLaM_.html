jaya maMgaLaM

English Lyrics

Composer: tyAgarAja
Raga: nadanamakriya
Tala: Adi
Language: telugu

Pallavi:
jaya maMgaLaM nitya subha maMgaLaM

Anupallavi:

Charanam:
1. maMgaLaM avani sutA nAthuniki
maMgaLaM aravindAkShuniki
maMgaLaM adbhuta cAritruniki
maMgaLaM Adi dEvuniki (jaya)

2. sundara vadanuniki su-dEhuniki
bRndAraka gaNa vandyuniki
mandara dharuniki mana
mAdhavuniki shubha phaladuniki (jaya)

3. ina kulamuna velasina rAmuniki
janaka vacana paripAluniki
manasija kOTi lAvaNyuniki
kanaka siMhAsana nilayuniki (jaya)

4. mandAnila bhOja shayanuniki
mandAkinI vara janakuniki
manda janaka shata saMkAshuniki
mandAra rUpuniki hariki (jaya)


5. indrAdyaShTa dig-Isha nutuniki
candrAditya su-nayanuki
gajEndruni saMrakShincina rAma
candruniki jagad-rUpuniki (jaya)

6. rAja shEkhara priyuniki mauni
rAja rAja pUjituniki tyAga-
rAja vinutuniki vara gaja
rAjAdi bhakta varaduniki (jaya)

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ನಾದನಾಮಕ್ರಿಯಾ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಜಯ ಮಂಗಳಂ ನಿತ್ಯ ಸುಭ ಮಂಗಳಂ

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
1. ಮಂಗಳಂ ಅವನಿ ಸುತಾ ನಾಥುನಿಕಿ
ಮಂಗಳಂ ಅರವಿಂದಾಕ್ಷುನಿಕಿ
ಮಂಗಳಂ ಅದ್ಭುತ ಚಾರಿತ್ರುನಿಕಿ
ಮಂಗಳಂ ಆದಿ ದೇವುನಿಕಿ (ಜಯ)

2. ಸುಂದರ ವದನುನಿಕಿ ಸು-ದೇಹುನಿಕಿ
ಬೃಂದಾರಕ ಗಣ ವಂದ್ಯುನಿಕಿ
ಮಂದರ ಧರುನಿಕಿ ಮನ
ಮಾಧವುನಿಕಿ ಶುಭ ಫಲದುನಿಕಿ (ಜಯ)

3. ಇನ ಕುಲಮುನ ವೆಲಸಿನ ರಾಮುನಿಕಿ
ಜನಕ ವಚನ ಪರಿಪಾಲುನಿಕಿ
ಮನಸಿಜ ಕೋಟಿ ಲಾವಣ್ಯುನಿಕಿ
ಕನಕ ಸಿಂಹಾಸನ ನಿಲಯುನಿಕಿ (ಜಯ)

4. ಮಂದಾನಿಲ ಭೋಜ ಶಯನುನಿಕಿ
ಮಂದಾಕಿನೀ ವರ ಜನಕುನಿಕಿ
ಮಂದ ಜನಕ ಶತ ಸಂಕಾಶುನಿಕಿ
ಮಂದಾರ ರೂಪುನಿಕಿ ಹರಿಕಿ (ಜಯ)


5. ಇಂದ್ರಾದ್ಯಷ್ಟ ದಿಗ್-ಈಶ ನುತುನಿಕಿ
ಚಂದ್ರಾದಿತ್ಯ ಸು-ನಯನುಕಿ
ಗಜೇಂದ್ರುನಿ ಸಂರಕ್ಷಿಂಚಿನ ರಾಮ
ಚಂದ್ರುನಿಕಿ ಜಗದ್-ರೂಪುನಿಕಿ (ಜಯ)

6. ರಾಜ ಶೇಖರ ಪ್ರಿಯುನಿಕಿ ಮೌನಿ
ರಾಜ ರಾಜ ಪೂಜಿತುನಿಕಿ ತ್ಯಾಗ-
ರಾಜ ವಿನುತುನಿಕಿ ವರ ಗಜ
ರಾಜಾದಿ ಭಕ್ತ ವರದುನಿಕಿ (ಜಯ)
