parA shaktiM

English Lyrics

Composer: muthuswami dikshitar
Raga: rudrapriya
Tala: Adi
Language: sanskrit

Pallavi:
parA shaktiM bhajarE pAmara

Anupallavi:

Charanam:
paramArtha rUpiNIM pApa vimOcanIM
bhakta jana pAlinIM bhaNDa mardinIM
guru guha rUpiNIM guNa traya rUpiNIM
guru maNDala vAsinIM kauLinIm

Kannada Lyrics

ರಚನೆ: ಮುತ್ತುಸ್ವಾಮಿ ದೀಕ್ಷಿತರ್
ರಾಗ: ರುದ್ರ ಪ್ರಿಯ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ಸಂಸ್ಕೃತ

ಪಲ್ಲವಿ:
ಪರಾ ಶಕ್ತಿಂ ಭಜರೇ ಪಾಮರ

ಚರಣ:

ಗುರು ಗುಹ ರೂಪಿಣೀಂ ಗುಣ ತ್ರಯ ರೂಪಿಣೀಂ
