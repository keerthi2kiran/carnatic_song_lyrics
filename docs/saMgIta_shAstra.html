ಸಂಗೀತ ಶಾಸ್ತ್ರ (saMgIta shAstra )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಮುಖಾರಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಸಂಗೀತ ಶಾಸ್ತ್ರ ಜ್ನಾನಮು
ಸಾರೂಪ್ಯ ಸೌಖ್ಯದಮೇ ಮನಸಾ

ಅನುಪಲ್ಲವಿ:
ಶೃಮ್ಗಾರ ರಸಾದ್ಯಖಿಲ
ಪೂರಿತ ರಾಮ ಕಥಾನಂದಾಬ್ಧಿ ಯುತ (ಸಂಗೀತ)

ಚರಣ:
3 ಪ್ರೇಮ ಭಕ್ತಿ ಸು-ಜನ ವಾತ್ಸಲ್ಯಮು
ಶ್ರೀಮದ್ ರಮಾ ವರ ಕಟಕ್ಷಮು
ನೇಮ ನಿಷ್ಠ ಯಶೋ ಧನಮೊಸಂಗುನೇ
ನೇರ್ಪು ಗಲ್ಗು ತ್ಯಾಗರಾಜು ನೇರ್ಚಿನ (ಸಂಗೀತ)

English Lyrics

Composer: tyAgarAja
Raga: mukhari
Tala: Adi
Language: telugu

Pallavi:
saMgIta shAstra jnAnamu
sArUpya saukhyadamE manasA

Anupallavi:
shRMgAra rasAdyakhila
pUrita rAma kathAnandAbdhi yuta (saMgIta)

Charanam:
3 prEma bhakti su-jana vAtsalyamu
shrImad ramA vara kaTakShamu
nEma niShTha yashO dhanamosaMgunE
nErpu galgu tyAgarAju nErcina (saMgIta)
