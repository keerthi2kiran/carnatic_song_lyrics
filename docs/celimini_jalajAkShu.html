ಚೆಲಿಮಿನಿ ಜಲಜಾಕ್ಷು (celimini jalajAkShu )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಯದುಕುಲ ಕಾಂಭೋಜಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಚೆಲಿಮಿನಿ ಜಲಜಾಕ್ಷು ಕಂಟೇ ಚೆಪ್ಪರಯ್ಯಾ ಮೀರು

ಅನುಪಲ್ಲವಿ:
ಪಲುಮಾರು ಮ್ರೊಕ್ಕೆದನು ದಯತೋ ಪಲುಕರಯ್ಯಾ ಎಂತೋ (ಚೆ)

ಚರಣ:
1. ಶರ ಚಾಪಮು ಕರಮುನನಿಡಿ ಮೆರಯುನಯ್ಯಾ
ಎಂತೋ ಕರುಣಾ ರಸಮು ನಿಂಡಿನ ಕನ್ನುಲಯ್ಯಾ (ಚೆ)

2. ಜೂಡ ಜೂಡ ಮನಸು ಕರಗೆಡು ಸುಮುಖುಡಯ್ಯಾ
ಭಕ್ತುಲ ಜಾಡ ತೆಲಿಸಿ ಮಾಟಲಾಡೆಡು ಜಾಣುಡಯ್ಯಾ (ಚೆ)


3. ಶೃಮ್ಗಾರುನಿ ಬಾಸಿ ಮೇನು ಚಿಕ್ಕೆನಯ್ಯಾ ಹರಿ
ಚೆಂಗಟ ಮುನ್ನೇ ನಾ ಮದಿ ಚಿಕ್ಕೆನಯ್ಯಾ (ಚೆ)


4. ನಾ ಲೋನಿ ಜಾಲಿನಿ ಪಲ್ಕ ಜಾಲನಯ್ಯಾ ಹರಿ
ಮೀಲೋ ಮೀರೇ ತೆಲಿಸಿ ಮರ್ಮಮಿಯ್ಯರಯ್ಯಾ (ಚೆ)

5. ತ್ಯಾಗರಾಜ ಸಖುಡನಿ ತಲತುನಯ್ಯಾ ಮೀರು
ಬಾಗುಗ ನಾಯಂಗಲಾರ್ಪು ಬಾಪರಯ್ಯಾ (ಚೆ)

English Lyrics

Raga: yadukula kambhoji
Tala: Adi
Language: telugu

Pallavi:
celimini jalajAkShu kaNTE cepparayyA mIru

Anupallavi:
palumAru mrokkedanu dayatO palukarayyA entO (ce)

Charanam:
1. shara cApamu karamunaniDi merayunayyA
entO karuNA rasamu niNDina kannulayyA (ce)

2. jUDa jUDa manasu karageDu sumukhuDayyA
bhaktula jADa telisi mATalADeDu jANuDayyA (ce)


3. shRMgAruni bAsi mEnu cikkenayyA hari
ceMgaTa munnE nA madi cikkenayyA (ce)


4. nA lOni jAlini palka jAlanayyA hari
mIlO mIrE telisi marmamiyyarayyA (ce)

5. tyAgarAja sakhuDani talatunayyA mIru
bAguga nAyaMgalArpu bAparayyA (ce)
