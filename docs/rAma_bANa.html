ರಾಮ ಬಾಣ (rAma bANa )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಸಾವೇರಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ರಾಮ ಬಾಣ ತ್ರಾಣ ಶೌರ್ಯಮ್-
ಏಮನಿ ತೆಲುಪುದುರಾ ಓ ಮನಸಾ

ಅನುಪಲ್ಲವಿ:
ಭಾಮಕಾಸಪಡು ರಾವಣ ಮೂಲ
ಬಲಮುಲ ನೇಲ ಕೂಲ ಜೇಯು (ರಾಮ)

ಚರಣ:
ತಮ್ಮುಡು ಬಡಲಿನ ವೇಳ ಸುರ ರಿಪು
ತೆಮ್ಮನಿ ಚಕ್ಕೆರ ಪಂಚೀಯಗ ಕನಿ
ಲೆಮ್ಮನುಚುನು ಇಂದ್ರಾರಿ ಪಲ್ಕ
ಸಮಯಮ್ಮನಿ ಲೇವಗಾ
ಸಮ್ಮತಿತೋ ನಿಲಬಡಿ ಕೋದಂಡಪು
ಜ್ಯಾ ಘೋಷಮುಲಶನುಲ ಜೇಸಿ ತಾ
ನೆಮ್ಮದಿ-ಗಲ ತೋಡುನು ಜೂಚೆನುರಾ
ನಿಜಮೈನ ತ್ಯಾಗರಾಜ ನುತುಡಗು (ರಾಮ)
