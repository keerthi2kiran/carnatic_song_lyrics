ಚಲಮೇಲ ಜೇಸೇವುರಾ ಸರಸಗು (calamEla jEsEvurA sarasagu )

Kannada Lyrics

ರಚನೆ: ಸ್ವಾತಿ ತಿರುನಾಳ್
ರಾಗ: ಶಂಕರಾಭರಣ
ತಾಳ: ಅಟ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಚಲಮೇಲ ಜೇಸೇವುರಾ
ಸರಸಗು ರಾರಾ

ಅನುಪಲ್ಲವಿ:
ಜಲಜಾಕ್ಷುಡಯಿನ ಚಕ್ಕನಿ
ಸರಸಿಜನಾಭ ದೇವಾ

ಚರಣ:
ಪಾಪ ಜಾಡಿ ಮಾರುಡು

English Lyrics
