ರಾರಾ ರಘುವೀರ (rArA raghuvIra )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಅಠಾಣ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ರಾರಾ ರಘುವೀರ ವೆಂಟ ರಾರಾ ತೋಡು

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
1. ಅನುದಿನಮು ನಿನು ಮನಸುನ ಕನುಗೊನಿ
ಆನಂದಮಾಯೆ ದಯಾಳೋ (ರಾರಾ)

2. ಸಕಲ ಸುಜನುಲು ಕೊಲುಚು ಸನ್ನಿಧಿ ಕನಿ
ಚಲ್ಲನಾಯೆ ದಯಾಳೋ (ರಾರಾ)

3. ಪಲು ವಿಧ ಚೆಡು ದುರ್-ವಿಷಯ ಚಯಮುಲ್-
ಎಡಬಾಯನಾಯೆ ದಲಾಳೋ (ರಾರಾ)

4. ತೊಡರಿಯಡುಗಡುಗುಕಿದಿ ಬುದ್ಧಿಯನಿ
ಸಂತೋಷಮಾಯೆ ದಯಾಳೋ (ರಾರಾ)

5. ಸುಮುಖಮುನನು ತಿಲಕಮು ಚೆಲಗಗ ಕನಿ
ಸೊಕ್ಕನಾಯೆ ದಯಾಳೋ (ರಾರಾ)

6. ಕವಗೊನಿ ಭವಮುನ ಪೊರಲಕ ನೀದು
ಕಾರ್ಯಮಾಯೆ ದಯಾಳೋ (ರಾರಾ)

7. ದಶರಥ ತನಯ ಶುಭ ಚರಿತ ಪಾಲಿತ
ತ್ಯಾಗರಾಜ ದಯಾಳೋ (ರಾರಾ)

English Lyrics

Composer: tyAgarAja
Raga: athana
Tala: Adi
Language: telugu

Pallavi:
rArA raghuvIra veNTa rArA tODu

Anupallavi:

Charanam:

callanAye dayALO (rArA)

3. palu vidha ceDu dur-viShaya cayamul-
eDabAyanAye dalALO (rArA)

4. toDariyaDugaDugukidi buddhiyani
santOShamAye dayALO (rArA)

5. sumukhamunanu tilakamu celagaga kani
sokkanAye dayALO (rArA)

6. kavagoni bhavamuna poralaka nIdu
kAryamAye dayALO (rArA)

7. dasharatha tanaya shubha carita pAlita
tyAgarAja dayALO (rArA)
