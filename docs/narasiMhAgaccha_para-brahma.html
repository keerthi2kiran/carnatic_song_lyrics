ನರಸಿಂಹಾಗಚ್ಛ ಪರ-ಬ್ರಹ್ಮ (narasiMhAgaccha para-brahma )

Kannada Lyrics

ರಚನೆ: ಮುತ್ತುಸ್ವಾಮಿ ದೀಕ್ಷಿತರ್
ರಾಗ: ಮೋಹನ
ತಾಳ: ಛಾಪು
ಭಾಷೆ: ಸಂಸ್ಕೃತ

ಪಲ್ಲವಿ:
ನರಸಿಂಹಾಗಚ್ಛ
ಪರ-ಬ್ರಹ್ಮ ಪುಚ್ಛ ಸ್ವೇಚ್ಛ ಸ್ವಚ್ಛ

ಅನುಪಲ್ಲವಿ:
2 ಹರಿ ಹರ ಬ್ರಹ್ಮೇಂದ್ರಾದಿ ಪೂಜಿತಾತ್ಯಚ್ಛ
ಪರಮ ಭಾಗವತ ಪ್ರಹ್ಲಾದ ಭಕ್ತ್ಯಚ್ಛ

ಚರಣ:
ಧೀರ-ತರ ಘಟಿಕಾಚಲೇಶ್ವರ
ಸೌರ-ತರ ಹೇಮ ಕೋಟೀಶ್ವರ
ವೀರ ವರ ಮೋಹನ ವಿಭಾಸ್ವರ
ಮಾರ ವರ ಮಾನವ ಹರೀಶ್ವರ
(ಮಧ್ಯಮ ಕಾಲ ಸಾಹಿತ್ಯಮ್)
ಮುರ ಹರ ನಗ ಧರ ಸರಸಿಜ ಕರ
ಪರಮ ಪುರುಷ ಪವನಜ ಶುಭ-ಕರ
ಸುರುಚಿರ ಕರಿ ಗಿರಿ ವರದ ವಿಚರ
ಸರಸ ಗುರು ಗುಹ ಹೃದಯ ಸಹ-ಚರ
