ಇಂದುಕಾಯೀ ತನುವುನು (indukAyI tanuvunu )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಮುಖಾರಿ
ತಾಳ: ಮಿಶ್ರ ಛಾಪು
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಇಂದುಕಾಯೀ ತನುವುನು ಪೆಂಚಿನ(ದಿ)

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
1. ನೀ ಸೇವಕು ಲೇಕ ನೀದು ಚೆಂತಕು ರಾಕ
ಆಸ ದಾಸುಡೈ ಅಟುಯಿಟು ತಿರಿಗುಟ(ನ್ದುಕಾ)


2. ನಿರತಮು ನೀ ದೃಷ್ಟಿ ನೇನಾರ್ಜಿಂಚಕ
ಓರುಲ ಭಾಮಲನು ಓರ ಜೂಪುಲು ಜೂಚುಟ(ನ್ದುಕಾ)

3. ಸಾರೆಕು ನಾಮ ಸ್ಮರಣಮು ಸೇಯಕ
ಊರಿ ಮಾಟಲೆಲ್ಲನೂರಕ ವದರುಟ(ನ್ದುಕಾ)

4. ಕರಮುಲತೋ ಪೂಜ ಗಾವಿಂಪಗ ದಾಚಿ
ಧರಲೋನ ಲೇನಿ ದುರ್-ದಾನಮುಲಕು ಚಾಚುಟ(ನ್ದುಕಾ)


5. ವಾರಮು ನೀ ಕ್ಷೇತ್ರ ವರಮುಲ ಜುಟ್ಟಕ
ಭೂರಿಕಿ ಮುಂದುಗಾ ಪಾರಿ ಪಾರಿ ತಿರುಗುಟ(ನ್ದುಕಾ)


6. ನೀವಾಡನು ಪೇರುನಿಂದು ವಹಿಂಚಕ
ನಾವಾಡನಿ ಯಮುಡು ನವ್ವುಚು ಬಾಧಿಂಚುಟ(ನ್ದುಕಾ)


7. ರಾವಯ್ಯ ಶ್ರೀ ತ್ಯಾಗರಾಜ ವಿನುತ ನಿನ್ನು
ಭಾವಿಂಚಕ ಪ್ರೊದ್ದು ಪಾರ ಗೊಟ್ಟುಕೊನ(ನ್ದುಕಾ)

English Lyrics

Composer: tyAgarAja
Raga: mukhari
Tala: miSra chApu
Language: telugu

Pallavi:
indukAyI tanuvunu pencina(di)

Anupallavi:

Charanam:
1. nI sEvaku lEka nIdu centaku rAka
Asa dAsuDai aTuyiTu tiriguTa(ndukA)


2. niratamu nI dRShTi nEnArjincaka
orula bhAmalanu Ora jUpulu jUcuTa(ndukA)

3. sAreku nAma smaraNamu sEyaka
Uri mATalellanUraka vadaruTa(ndukA)

4. karamulatO pUja gAvimpaga dAci
dharalOna lEni dur-dAnamulaku cAcuTa(ndukA)


5. vAramu nI kShEtra varamula juTTaka
bhUriki mundugA pAri pAri tiruguTa(ndukA)


6. nIvADanu pErunindu vahincaka
nAvADani yamuDu navvucu bAdhincuTa(ndukA)


7. rAvayya shrI tyAgarAja vinuta ninnu
bhAvincaka proddu pAra goTTukona(ndukA)
