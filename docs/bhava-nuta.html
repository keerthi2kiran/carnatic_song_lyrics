ಭವ-ನುತ (bhava-nuta )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಮೋಹನ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಭವ-ನುತ ನಾ ಹೃದಯಮುನ ರಮಿಂಪುಮು
ಬಡಲಿಕ ತೀರ

ಅನುಪಲ್ಲವಿ:
ಭವ ತಾರಕ ನಾತೋ ಬಹು ಪಲ್ಕಿನ
ಬಡಲಿಕ ತೀರ ಕಮಲ ಸಮ್-(ಭವ)

ಚರಣ:
1.
ಪವನ ಸುತ ಪ್ರಿಯ
ತನಕೈ ತಿರಿಗಿನ ಬಡಲಿಕ ತೀರ
ಭವನಮು ಜೇರಿ ನನು ವೆರಪಿಂಚಿನ
ಬಡಲಿಕ ತೀರ ಕಮಲ ಸಮ್-(ಭವ)3.
ವರಮಗು ನೈವೇದ್ಯಮುಲನು
ಜೇಯನಿ ಬಡಲಿಕ ತೀರ
ಪರವ ಲೇಕ ಸರಿ ಪೋಯಿನಟ್ಟಾಡಿನ
ಬಡಲಿಕ ತೀರ ಕಮಲ ಸಮ್-(ಭವ)5.
ಪ್ರಬಲ ಜೇಸಿ ನನು ಬ್ರೋಚೆದ-ವನುಕೊನ್ನ
ಬಡಲಿಕ ತೀರ
ಪ್ರಭು ನೀವು ತ್ಯಾಗರಾಜುನಿಕಿ
ಬಡಲಿಕ ತೀರ ಕಮಲ ಸಮ್-(ಭವ)
