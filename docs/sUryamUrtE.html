ಸೂರ್ಯಮೂರ್ತೇ (sUryamUrtE )

Kannada Lyrics

ರಚನೆ: ಮುತ್ತುಸ್ವಾಮಿ ದೀಕ್ಷಿತರ್
ರಾಗ: ಸೌರಾಷ್ಟ್ರ
ತಾಳ: ಧ್ರುವ
ಭಾಷೆ: ಸಂಸ್ಕೃತ

ಪಲ್ಲವಿ:
ಸೂರ್ಯಮೂರ್ತೇ ನಮೋಸ್ತು ತೇ
ಸುಂದರ ಛಾಯಾಧಿಪತೇ

ಅನುಪಲ್ಲವಿ:
ಕಾರ್ಯ ಕಾರಣಾತ್ಮಕ ಜಗದ್ಪ್ರಕಾಶ -
ಸಿಂಹ ರಾಶ್ಯಧಿಪತೇ
(ಮಧ್ಯಮ ಕಾಲ ಸಾಹಿತ್ಯಮ್)
ಆರ್ಯ ವಿನುತ ತೇಜಸ್ಸ್ಫೂರ್ತೇ
ಆರೋಗ್ಯಾದಿ ಫಲದ ಕೀರ್ತೇ

ಚರಣ:
ಸಾರಸ ಮಿತ್ರ ಮಿತ್ರ ಭಾನೋ
ಸಹಸ್ರ ಕಿರಣ ಕರ್ಣ ಸೂನೋ
ಕ್ರೂರ ಪಾಪ ಹರ ಕೃಶಾನೋ
ಗುರು ಗುಹ ಮೋದಿತ ಸ್ವಭಾನೋಸೂರಿ ಜನೇಡಿತ ಸುದಿನಮಣೇ
ಸೋಮಾದಿ ಗ್ರಹ ಶಿಖಾಮಣೇ
ಧೀರಾರ್ಚಿತ ಕರ್ಮ ಸಾಕ್ಷಿಣೇ
ದಿವ್ಯತರ ಸಪ್ತಾಶ್ವ ರಥಿನೇಸೌರಾಷ್ಟಾರ್ಣ ಮಂತ್ರಾತ್ಮನೇ
ಸೌವರ್ಣ ಸ್ವರೂಪಾತ್ಮನೇ
ಭಾರತೀಶ ಹರಿ ಹರಾತ್ಮನೇ
ಭುಕ್ತಿ ಮುಕ್ತಿ ವಿತರಣಾತ್ಮನೇ

Raga: saurashtram
Tala: Dhruvam
Language: sanskrit
