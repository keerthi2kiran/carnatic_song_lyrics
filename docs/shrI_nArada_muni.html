ಶ್ರೀ ನಾರದ ಮುನಿ (shrI nArada muni )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಭೈರವಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಶ್ರೀ ನಾರದ ಮುನಿ ಗುರು ರಾಯ ಕಂಟಿ-
ಮೇ ನಾಟಿ ತಪಮೋ ಗುರು ರಾಯ

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
1. ಮನಸಾರ ಕೋರಿತಿ ಗುರು ರಾಯ ನೇಡು
ಕನುಲಾರ ಕನುಕೊಂಟಿಮಿ ಗುರು ರಾಯ (ಶ್ರೀ)

2. ಮೀ ಸೇವ ದೊರಿಕೆನು ಗುರು ರಾಯ ಭವ
ಪಾಶಮು ತೊಲಗೆನು ಗುರು ರಾಯ (ಶ್ರೀ)


3. ನೀವೇ ಸು-ಜ್ನಾನ ಸುಖಿ ಗುರು ರಾಯ
ನೀವೇಯಜ್ನಾನ ಶಿಖಿ ಗುರು ರಾಯ (ಶ್ರೀ)


4. ರಾಜಿಲ್ಲು ವೀಣೆ ಗಲ ಗುರು ರಾಯ ತ್ಯಾಗ-
ರಾಜುನಿ ಬ್ರೋಚಿನ ಸದ್-ಗುರು ರಾಯ (ಶ್ರೀ)
