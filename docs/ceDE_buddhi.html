ಚೆಡೇ ಬುದ್ಧಿ (ceDE buddhi )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಅಠಾಣ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಚೆಡೇ ಬುದ್ಧಿ ಮಾನುರಾ

ಅನುಪಲ್ಲವಿ:
ಇಡೇ ಪಾತ್ರಮೆವರೋ ಜೂಡರಾ (ಚೆಡೇ)

ಚರಣ:
ಭೂ ವಾಸಿಕಿ ತಗು ಫಲಮು ಕಲ್ಗುನನಿ
ಬುಧುಲು ಪಲ್ಕ ವಿನ ಲೇದಾ ಮನಸಾ
ಶ್ರೀ ವಾಸುದೇವಸ್-ಸರ್ವಮನುಚುನು
ಚಿಂತಿಂಚರಾ ತ್ಯಾಗರಾಜ ವಿನುತುನಿ (ಚೆಡೇ)
