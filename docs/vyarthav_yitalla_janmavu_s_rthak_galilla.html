ವ್ಯರ್ಥವಾಯಿತಲ್ಲ ಜನ್ಮವು ಸಾರ್ಥಕಾಗಲಿಲ್ಲ (vyarthavāyitalla janmavu sārthakāgalilla)

Kannada Lyrics

ರಚನೆ: ಕನಕ ದಾಸರು
ರಾಗ:
ತಾಳ:
ಭಾಷೆ: ಕನ್ನಡ

ಪಲ್ಲವಿ:

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
ವ್ಯರ್ಥವಾಯಿತಲ್ಲ ಜನ್ಮವು ಸಾರ್ಥಕಾಗಲಿಲ್ಲ
ಅರ್ಥದಾಸೆಯಲಿ ಪೃಥ್ವಿಯೊಳಗೆ ಸುತ್ತಿ
ಮತ್ತನಾಗಿ ಬಲು ಕೃತ್ಯಗಳನು ಮಾಡಿ
ಹರಿಯ ನೆನೆಯಲಿಲ್ಲ ಹರುಷದಿ ಗರುವ ಪುಟ್ಟಿತಲ್ಲ
ಪರಮ ಮೂಢರಲಿ ನಿರುತ ಸಂಗವ ಮಾಡಿ
ಗುರುಹಿರಿಯರ ದಯೆ ದೊರೆಯದೆ ಸುಮ್ಮನೆ ||1||
ಏನು ಪೇಳಲೇನು ಎನಗೆ ಹೀನ ಬಿಡದು ಮುನ್ನ
ಶ್ವಾನಗಿಂತಲು ಬಲು ಮಾನಗೆಟ್ಟು ನಾ
ದೀನನಾಗಿ ಮನೆಮನೆಗಳ ತಿರುಗಿ ||2||
ಭಾಗವತರ ಪಾದಕ್ಕೊಂದಿನ ಬಾಗಿ ನಡೆಯಲಿಲ್ಲ
ರಾಗರಂಗು ಭಕ್ತಿಭಾವದೊಳು
ಬೀಗಿ ಚೆನ್ನಕೇಶವನನು ನೆನೆಯದೆ ||3||
