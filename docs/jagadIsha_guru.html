ಜಗದೀಶ ಗುರು (jagadIsha guru )

Kannada Lyrics

ರಚನೆ: ಮುತ್ತುಸ್ವಾಮಿ ದೀಕ್ಷಿತರ್
ರಾಗ: ಶಂಕರಾಭರಣ
ತಾಳ: ಚತುರಶ್ರ ಏಕ
ಭಾಷೆ: ಸಂಸ್ಕೃತ

ಪಲ್ಲವಿ:
ಜಗದೀಶ ಗುರು ಗುಹ ಹರಿ ವಿಧಿ ವಿನುತಂ
ದೇಹ-ತ್ರಯ ವಿಲಕ್ಷಣಂ ಆನಂದ ಲಕ್ಷಣಮ್
ನಿತ್ಯಂ ಶುದ್ಧಂ ಬುದ್ಧಂ ಮುಕ್ತಂ ಸತ್ಯಂ
ನಿರ್ವಿಕಲ್ಪಂ ನಿಷ್ಪ್ರಪಂಚಂ ಆನಂದಂ ಅಜಮ್
