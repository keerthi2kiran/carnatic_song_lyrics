pAhi mAM harE

English Lyrics

Composer: tyAgarAja
Raga: saurashtram
Tala: rupakam
Language: telugu

Pallavi:
pAhi mAM harE mahAnubhAva rAghava

Anupallavi:

Charanam:
1. pAhi rAmayanucu rEyu pagalu koliciti
pAhi rAma nIvanu sampadanu valaciti (pAhi)

2. pAhi rAmayanucu pAri pAri kOriti
pAhi rAma nAma muktA phalamulEriti (pAhi)

3. pAhi rAmayanucu bhuvini bAga puTTiti
pAhi rAmayanucu gaTTi paTTu paTTiti (pAhi)

4. pAhi rAmayanucu nIdu padamu nammiti
pAhi rAmayanucu manasu bAga grammiti (pAhi)

5. pAhi rAmayanucu nIdu padamu pADiti
pAhi rAmayanucu parama padamu vEDiti (pAhi)

6. pAhi rAmayanucu dhyAna paruDanaitini
pAhi tyAgarAja vinuta bhaktuDaitini (pAhi)

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಸೌರಾಷ್ಟ್ರ
ತಾಳ: ರೂಪಕ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಪಾಹಿ ಮಾಂ ಹರೇ ಮಹಾನುಭಾವ ರಾಘವ

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
1. ಪಾಹಿ ರಾಮಯನುಚು ರೇಯು ಪಗಲು ಕೊಲಿಚಿತಿ
ಪಾಹಿ ರಾಮ ನೀವನು ಸಂಪದನು ವಲಚಿತಿ (ಪಾಹಿ)

2. ಪಾಹಿ ರಾಮಯನುಚು ಪಾರಿ ಪಾರಿ ಕೋರಿತಿ
ಪಾಹಿ ರಾಮ ನಾಮ ಮುಕ್ತಾ ಫಲಮುಲೇರಿತಿ (ಪಾಹಿ)

3. ಪಾಹಿ ರಾಮಯನುಚು ಭುವಿನಿ ಬಾಗ ಪುಟ್ಟಿತಿ
ಪಾಹಿ ರಾಮಯನುಚು ಗಟ್ಟಿ ಪಟ್ಟು ಪಟ್ಟಿತಿ (ಪಾಹಿ)

4. ಪಾಹಿ ರಾಮಯನುಚು ನೀದು ಪದಮು ನಮ್ಮಿತಿ
ಪಾಹಿ ರಾಮಯನುಚು ಮನಸು ಬಾಗ ಗ್ರಮ್ಮಿತಿ (ಪಾಹಿ)

5. ಪಾಹಿ ರಾಮಯನುಚು ನೀದು ಪದಮು ಪಾಡಿತಿ
ಪಾಹಿ ರಾಮಯನುಚು ಪರಮ ಪದಮು ವೇಡಿತಿ (ಪಾಹಿ)

6. ಪಾಹಿ ರಾಮಯನುಚು ಧ್ಯಾನ ಪರುಡನೈತಿನಿ
ಪಾಹಿ ತ್ಯಾಗರಾಜ ವಿನುತ ಭಕ್ತುಡೈತಿನಿ (ಪಾಹಿ)
