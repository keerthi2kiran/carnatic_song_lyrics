ಶ್ರೀ ರಘುವರಾಪ್ರಮೇಯ ಮಾಮವ (shrI raghuvarApramEya mAmava )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಕಾಂಭೋಜಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಶ್ರೀ ರಘುವರಾಪ್ರಮೇಯ ಮಾಮವ

ಅನುಪಲ್ಲವಿ:
ಶ್ರೀ ರಘು ಕುಲ ಜಲ ನಿಧಿ ಸೋಮ
ಶ್ರೀ ರಾಮ ಪಾಲಯ (ಶ್ರೀ)

ಚರಣ:
1. ಸಾರಸ ಹಿತ ಕುಲಾಬ್ಜ ಭೃಮ್ಗ
ಸಂಗೀತ ಲೋಲ (ಶ್ರೀ)

2. ವಿರೋಚನ ಕುಲೇಶ್ವರ ಸ್ವರ ಲಯಾದಿ
ಮೂರ್ಚನೋಲ್ಲಸಿತ ನಾರದ ವಿನುತ (ಶ್ರೀ)


3. ಶ್ರೀ ಭಾಸ್ಕರ ಕುಲಾದ್ರಿ ದೀಪ
ಶ್ರೀ ಭಾಗವತ ವಿನುತ ಸು-ಚರಣ (ಶ್ರೀ)

4. ಸೀತಾ ನಾಥ ತ್ಯಾಗರಾಜ
ನುತಾನಿಲ ಸುತಾಪ್ತ ಸುಗುಣಾಭರಣ (ಶ್ರೀ)
