ಅನ್ಯಾಯಮು (anyAyamu )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಕಾಪಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಅನ್ಯಾಯಮು ಸೇಯಕುರಾ ರಾಮ ನನ್ನ್-
ಅನ್ಯುನಿಗಾ ಜೂಡಕುರಾ ನಾಯೆಡ ರಾಮ

ಅನುಪಲ್ಲವಿ:
2 ಎನ್ನೋ ತಪ್ಪುಲು ಗಲ ವಾರಿನಿ
ರಾಜನ್ಯ ನೀವು ಬ್ರೋಚಿನಾವು ಗನುಕನು (ಅನ್ಯಾಯ)

ಚರಣ:
3 ಜಡ ಭರತುಡು ಜಿಂಕ ಶಿಶುವುನೆತ್ತಿ
ಬಡಲಿಕ ತೀರ್ಚಗ ಲೇದಾ
ಕಡಲಿನಿ ಮುನಿಗಿನ ಗಿರಿನೊಕ
ಕೂರ್ಮಮು ಕಾಪಾಡಗ ಲೇದಾ
ಪುಡಮಿನಿ ಪಾಂಡವ ದ್ರೋಹಿನಿ
ಧರ್ಮ ಪುತ್ರುಡು ಬ್ರೋವಗ ಲೇದಾ
ನಡಿಮಿ ಪ್ರಾಯಮುನ ತ್ಯಾಗರಾಜ ನುತ
ನಾ ಪೂರ್ವಜು ಬಾಧ ತೀರ್ಪ ಲೇನನಿ (ಅನ್ಯಾಯ)
