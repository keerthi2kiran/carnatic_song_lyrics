enta nErcina

English Lyrics

Composer: tyAgarAja
Raga: suddha dhanyasi - udaya ravichandrika
Tala: Adi
Language: telugu

Pallavi:
enta nErcina enta jUcina
enta vAralaina kAnta dAsulE

Anupallavi:
santatambu shrI kAnta swAnta
siddhAntamaina mArga cinta lEni vA-(renta)

Charanam:
para himsa para bhAmAnya dhana
para mAnavApavAda
para jIvanAdulakanRtamE
bhAShincErayya tyAgarAja nuta (enta)

Kannada Lyrics
