ಪಲುಕು ಕಂಡ (paluku kaNDa )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ನವರಸ ಕನ್ನಡ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಪಲುಕು ಕಂಡ ಚಕ್ಕೆರನು ಕೇರುನೇ
ಪಣತುಲಾರ ಜೂಡರೇ

ಅನುಪಲ್ಲವಿ:
ಕೊಲುವು ಕೂಟಮುನನು ಕೊಲುಚು ವಾರಿನಿ
ಪಿಲಿಚಿ ದಾಶರಥಿ ಪ್ರೇಮ ಮೀರ ಪಲಿಕೇ (ಪಲುಕು)

ಚರಣ:
ಸುರುಲ ಕಾಮಿನೀ ಮಣುಲ ಗಾನಮ್-
ಆದರಣನಾಲಕಿಂಚುಚುನು ಶೃಮ್ಗಾರ
ರಸ ಯುಕ್ತ ವಾರ ರಮಣುಲಾಡ ಜೂಚಿ
ಸರಸ ತ್ಯಾಗರಾಜುನಿತೋನು ಪೊಗಡೇ (ಪಲುಕು)
