ರಾಜ ರಾಜೇಂದ್ರ ಚೋಳ (rAja rAjEndra cOLa )

Kannada Lyrics

ರಚನೆ: ಮುತ್ತುಸ್ವಾಮಿ ದೀಕ್ಷಿತರ್
ರಾಗ: ಗುಂಡಕ್ರಿಯ
ತಾಳ: ತ್ರಿಪುಟ
ಭಾಷೆ: ಸಂಸ್ಕೃತ

ಪಲ್ಲವಿ:
ರಾಜ ರಾಜೇಂದ್ರ ಚೋಳ ಪ್ರತಿಷ್ಠಿತಂ
ಬೃಹದೀಶ್ವರಂ ಭಜ ರೇ ಶ್ರೀ

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
ರಾಜಾಧಿ-ರಾಜ ಸಮಾರ್ಚಿತಂ
ರಮಣೀಯ ಹೃದಯ ವಿರಾಜಿತಂ
ರಾಜೀವ ಲೋಚನಂ ಗುರು ಗುಹ -
ರಂಜಿತ ಗುಂಡ ಕ್ರಿಯಾ ಪ್ರಿಯಂ
(ಮಧ್ಯಮ ಕಾಲ ಸಾಹಿತ್ಯಮ್)
ರಾಜ ರಾಜೇಂದ್ರ ಚೋಳ ಧರಂ
ರಾಜ ಧರಂ ಬೃಹದೀಶ್ವರಂ ಭಜ
