ಧ್ಯಾನಮೇ ವರಮೈನ (dhyAnamE varamaina )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಧನ್ಯಾಸಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಧ್ಯಾನಮೇ ವರಮೈನ ಗಂಗಾ ಸ್ನಾನಮೇ ಮನಸಾ

ಅನುಪಲ್ಲವಿ:
ವಾನ ನೀಟ ಮುನುಗ ಮುನುಗ ಲೋನಿ
ವಂಚನ ದ್ರೋಹಮನು ಕರ ಪೋನಾ (ಧ್ಯಾನ)

ಚರಣ:
ಪರ ಧನ ನಾರೀ ಮಣುಲನು ದೂರಿ
ಪರ ನಿಂದಲ ಪರ ಹಿಮ್ಸಲ ಮೀರಿ
ಧರನು ವೆಲಯು ಶ್ರೀ ರಾಮುನಿ ಕೋರಿ
ತ್ಯಾಗರಾಜು ತೆಲುಸುಕೊನ್ನ ರಾಮ (ಧ್ಯಾನ)

English Lyrics

Composer: tyAgarAja
Raga: dhanyasi
Tala: Adi
Language: telugu

Pallavi:
dhyAnamE varamaina gaMgA snAnamE manasA

Anupallavi:
vAna nITa munuga munuga lOni
vancana drOhamanu kara pOnA (dhyAna)

Charanam:
para dhana nArI maNulanu dUri
para nindala para himsala mIri
dharanu velayu shrI rAmuni kOri
tyAgarAju telusukonna rAma (dhyAna)
