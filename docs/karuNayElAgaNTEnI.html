ಕರುಣಯೇಲಾಗಂಟೇನೀ (karuNayElAgaNTEnI )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ವರಾಳಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಕರುಣಯೇಲಾಗಂಟೇನೀ ವಿಧಮೇ
ಕಲ್ಯಾಣ ಸುಂದರ ರಾಮ

ಅನುಪಲ್ಲವಿ:
1 ಪರಮಾತ್ಮುಡು ಜೀವಾತ್ಮುಡು ಓಕಡೈ
ಪರಗುಚುಂಡು ಭಕ್ತ ಪರಾಧೀನುನಿ (ಕರುಣ)

ಚರಣ:
1. ಅನೃತಂಬಾಡಡು ಅಲ್ಪುಲ ವೇಡಡು
ಸು-ನೃಪುಲ ಕೊಲುವಡು ಸೂರ್ಯುನಿ ಮರವಡು (ಕರುಣ)


2. ಮಾಮ್ಸಮು ಮುಟ್ಟಡು ಮಧುವುನು ತ್ರಾಗಡು
ಪರ-ಹಿಮ್ಸಲ ಸೇಯಡು ಎರುಕನು ಮರವಡು (ಕರುಣ)


3. ಮೂಡೀಷಣಮುಲ ವಾಡಡು ಜೀವನ್-
ಮುಕ್ತುಡೈ ತಿರುಗು ಮುದಮುನು ಜೂಪಡು (ಕರುಣ)


4. ವಂಚನ ಸೇಯಡು ವರುಲತೋ ಬೊಂಕಡು
ಚಂಚಲ ಚಿತ್ತುಡೈ ಸೌಖ್ಯಮು ವಿಡುವಡು (ಕರುಣ)



5. ಸಾಕ್ಷಿಯನಿ ತೆಲಿಸಿಯಂದು ಲಕ್ಷ್ಯಮು ವಿಡುವಡು
ಕಂಜಾಕ್ಷುನಿ ತ್ಯಾಗರಾಜ ರಕ್ಷಕುಡೈನ ವಾನಿ (ಕರುಣ)

English Lyrics

Composer: tyAgarAja
Raga: varali
Tala: Adi
Language: telugu

Pallavi:
karuNayElAgaNTEnI vidhamE
kalyANa sundara rAma

Anupallavi:
1 paramAtmuDu jIvAtmuDu okaDai
paragucuNDu bhakta parAdhInuni (karuNa)

Charanam:
1. anRtambADaDu alpula vEDaDu
su-nRpula koluvaDu sUryuni maravaDu (karuNa)


2. mAmsamu muTTaDu madhuvunu trAgaDu
para-himsala sEyaDu erukanu maravaDu (karuNa)


3. mUDIShaNamula vADaDu jIvan-
muktuDai tirugu mudamunu jUpaDu (karuNa)


4. vancana sEyaDu varulatO boMkaDu
cancala cittuDai saukhyamu viDuvaDu (karuNa)



5. sAkShiyani telisiyandu lakShyamu viDuvaDu
kanjAkShuni tyAgarAja rakShakuDaina vAni (karuNa)
