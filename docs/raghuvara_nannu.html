ರಘುವರ ನನ್ನು (raghuvara nannu )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಪಂತುವರಾಳಿ - ಕಾಮವರ್ಧಿನಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ರಘುವರ ನನ್ನು ಮರವ ತಗುನಾ

ಅನುಪಲ್ಲವಿ:
ನಗ ಧರ ಭಕ್ತ ಜನಾಘ ನಿವಾರಣ (ರಘು)

ಚರಣ:
1. ತಲ್ಲಿ ತಂಡ್ರುಲನ್ನ ತಮ್ಮುಲುಯುನ್ನಾ
ಪೊಲತಿಕಿಯೊಕಡು ಪುರುಷುಂಡೌನಾ (ರಘು)

2. ಪರ ದೈವಮುಲು ಬಾಗು ಸೊಮ್ಮುಲು
ಸುರ ನುತ ಮಂಗಳ ಸೂತ್ರಮುಲೌನಾ (ರಘು)

3. ಚೇಸಿನ ಪುಣ್ಯ ಚಯಮು ಬ್ರಹ್ಮಣ್ಯ
ಆಸನೊಸಂಗಿತಿ ಅನುಪಮ ಲಾವಣ್ಯ (ರಘು)

4. ಮನಸುನ ನೀಕೇ ಮರುಲುಕೊನ್ನಾನು
ಸಜ್ಜನ ಹಿತ ತ್ಯಾಗರಾಜ ನುತ ಶುಭಾ ಕರ (ರಘು)

English Lyrics

Composer: tyAgarAja
Raga: pantuvarali - kama vardhani
Tala: Adi
Language: telugu

Pallavi:
raghuvara nannu marava tagunA

Anupallavi:
naga dhara bhakta janAgha nivAraNa (raghu)

Charanam:
1. talli taNDrulanna tammuluyunnA
polatikiyokaDu puruShuNDaunA (raghu)

2. para daivamulu bAgu sommulu
sura nuta maMgaLa sUtramulaunA (raghu)

3. cEsina puNya cayamu brahmaNya
AsanosaMgiti anupama lAvaNya (raghu)

4. manasuna nIkE marulukonnAnu
sajjana hita tyAgarAja nuta shubhA kara (raghu)
