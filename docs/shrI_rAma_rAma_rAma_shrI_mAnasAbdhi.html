ಶ್ರೀ ರಾಮ ರಾಮ ರಾಮ ಶ್ರೀ ಮಾನಸಾಬ್ಧಿ (shrI rAma rAma rAma shrI mAnasAbdhi )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ನೀಲಾಂಬರಿ - ಗೋಪಿಕಾ ವಸಂತಂ
ತಾಳ: ಖಂಡ ಛಾಪು
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಶ್ರೀ ರಾಮ ರಾಮ ರಾಮ ಶ್ರೀ ಮಾನಸಾಬ್ಧಿ ಸೋಮ
ನಾರಾಯಣಾಪ್ತ ಕಾಮ ನಳಿನಾಕ್ಷ ಪವ್ವಳಿಂಚು (ಶ್ರೀ)

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
1. ಪಲ್ಲವಾಧರ ವರೇಣ್ಯ ಪಾಪೇಭ ಸಿಂಹ ಧಂಯ
ಮಲ್ಲಿಕಾ ತಲ್ಪಮಂದು ಮಾಧವ ಪವ್ವಳಿಂಚು (ಶ್ರೀ)

2. ಧಾರಾ ಧರಾಭ ದೇಹ ತಾರಾಧಿಪಾನನ ಸದಾ
ನೀ ರಾಕ ಕೋರಿಯುಂಟಿ ಶ್ರೀ ರಮ ಪವ್ವಲಿಂಚು (ಶ್ರಿ)

3. ಜನಕ ರಾಜಿಂಟ ಪುಟ್ಟಿ ಜಾನಕೀ ಚೆಟ್ಟ ಪಟ್ಟಿ
ಕನಕಮೌ ಸುರಟಿ ಪಟ್ಟಿ ಕಾಚಿನದಿ ಪವ್ವಲಿಂಚು (ಶ್ರೀ)

4. ವರ್ಣಿಂಪ ತರಮು ಕಾನಿ ಸ್ವರ್ಣಂಬು ಮೇನು ಸೀತಾ
ಪೂರ್ಣಮೌ ಭಕ್ತಿತೋನು ಪೂಜಿಂಚು ಪವ್ವಲಿಂಚು (ಶ್ರೀ)

5. ಆಶುಗಾಶನ ಸು-ಶಯನ ಅಂಭೋಜ ಪತ್ರ ನಯನ
ಆಶತೋ ತ್ಯಾಗರಾಜು ಅರ್ಚಿಂಚ ಪವ್ವಲಿಂಚು (ಶ್ರೀ)

Language: telugu

Pallavi:
shrI rAma rAma rAma shrI mAnasAbdhi sOma
nArAyaNApta kAma naLinAkSha pavvaLincu (shrI)

Anupallavi:

Charanam:

2. dhArA dharAbha dEha tArAdhipAnana sadA
nI rAka kOriyuNTi shrI rama pavvalincu (shri)

3. janaka rAjiNTa puTTi jAnakI ceTTa paTTi
kanakamau suraTi paTTi kAcinadi pavvalincu (shrI)

4. varNimpa taramu kAni svarNambu mEnu sItA
pUrNamau bhaktitOnu pUjincu pavvalincu (shrI)

5. AshugAshana su-shayana ambhOja patra nayana
AshatO tyAgarAju arcinca pavvalincu (shrI)
