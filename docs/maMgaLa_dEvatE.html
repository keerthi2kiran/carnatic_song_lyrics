ಮಂಗಳ ದೇವತೇ (maMgaLa dEvatE )

Kannada Lyrics

ರಚನೆ: ಮುತ್ತುಸ್ವಾಮಿ ದೀಕ್ಷಿತರ್
ರಾಗ: ಮಾರ್ಗ ದೇಶಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ಸಂಸ್ಕೃತ

ಪಲ್ಲವಿ:
ಮಂಗಳ ದೇವತೇ ಪರ ದೇವತೇ
ಮಂಗಳಂ ಭವತು ನತ ದೇವತೇ

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
ಅಂಗಜ ಪುರ ಕಾಲ ವೈರಿ ಸಹಿತೇ
ಅನಾದ್ಯವಿದ್ಯಾ ಪ್ರಪಂಚ ರಹಿತೇ
(ಮಧ್ಯಮ ಕಾಲ ಸಾಹಿತ್ಯಮ್)
ಪುಂಗವ ಗುರು ಗುಹಾದಿ ಮಹಿತೇ
ಸತ್ಸಂಗ ಮಾರ್ಗ ದರ್ಶಿತೇ ಸುರ ಹಿತೇ

English Lyrics

Composer: muthuswami dikshitar
Raga: marga desi
Tala: Adi
Language: sanskrit

Pallavi:
maMgaLa dEvatE para dEvatE
maMgaLaM bhavatu nata dEvatE

Anupallavi:

Charanam:
aMgaja pura kAla vairi sahitE
anAdyavidyA prapanca rahitE
(madhyama kAla sAhityam)
puMgava guru guhAdi mahitE
satsaMga mArga darshitE sura hitE
