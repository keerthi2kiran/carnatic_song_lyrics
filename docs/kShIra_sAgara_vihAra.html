ಕ್ಷೀರ ಸಾಗರ ವಿಹಾರ (kShIra sAgara vihAra )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಆನಂದ ಭೈರವಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ಸಂಸ್ಕೃತ

ಪಲ್ಲವಿ:
ಕ್ಷೀರ ಸಾಗರ ವಿಹಾರ ಅಪರಿಮಿತ
ಘೋರ ಪಾತಕ ವಿದಾರ
ಕ್ರೂರ ಜನ ಗಣ ವಿದೂರ ನಿಗಮ
ಸಂಚಾರ ಸುಂದರ ಶರೀರ

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
ಶತಮಖಾಹಿತ ವಿಭಂಗ ಶ್ರೀ ರಾಮ
ಶಮನ ರಿಪು ಸನ್ನುತಾಂಗ
ಶ್ರಿತ ಮಾನವಾಂತರಂಗ ಜನಕಜಾ
ಶೃಮ್ಗಾರ ಜಲಜ ಭೃಮ್ಗರಾಜಾಧಿ ರಾಜ ವೇಷ ಶ್ರೀ ರಾಮ
ರಮಣೀಯ ಕರ ಸುಭೂಷ
ರಾಜ ನುತ ಲಲಿತ ಭಾಷ ಶ್ರೀ ತ್ಯಾಗ-
ರಾಜಾದಿ ಭಕ್ತ ಪೋಷ
