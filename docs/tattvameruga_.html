tattvameruga

English Lyrics

Composer: tyAgarAja
Raga: garudadhwani
Tala: rupakam
Language: telugu

Pallavi:
tattvameruga taramA para

Anupallavi:
tattvamasiyanu vAkyArthamu
rAma nIvanu para (tattvameruga)

Charanam:
tAmasa rAjasa guNamula
tannukOLLu pOdayA
rAma bhaktuDaina tyAgarAja
vinuta vEda shAstra (tattvameruga)
