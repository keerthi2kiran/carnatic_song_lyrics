ಕನುಗೊನು (kanugonu )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ನಾಯಕಿ
ತಾಳ: ರೂಪಕ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಕನುಗೊನು ಸೌಖ್ಯಮು
ಕಮಲಜುಕೈನ ಗಲ್ಗುನಾ

ಅನುಪಲ್ಲವಿ:
ದನುಜ ವೈರಿಯಗು ರಾಮುನಿ
ದಯ ಗಲ್ಗಿನಯತನಿ ವಿನಾ (ಕನು)

ಚರಣ:
ತನುವೊಕಚೋ ಮನಸೊಕಚೋ
ತಗಿನ ವೇಷಮೊಕಚೋನಿಡಿ
ಜನುಲನೇಚು ವಾರಿಕಿ
ಜಯಮೌನೇ ತ್ಯಾಗರಾಜು (ಕನು)
