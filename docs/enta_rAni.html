ಎಂತ ರಾನಿ (enta rAni )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಹರಿ ಕಾಂಭೋಜಿ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಎಂತ ರಾನಿ ತನಕೆಂತ ಪೋನಿ ನೀ
ಚೆಂತ ವಿಡುವ ಜಾಲ ಶ್ರೀ ರಾಮ

ಅನುಪಲ್ಲವಿ:
3 ಅಂತಕಾರಿ ನೀ ಚೆಂತ ಜೇರಿ
ಹನುಮಂತುಡೈ ಕೊಲುವ ಲೇದಾ (ಎಂತ)

ಚರಣ:
1. ಶೇಷುಡು ಶಿವುನಿಕಿ ಭೂಷುಡು ಲಕ್ಷ್ಮಣ
ವೇಷಿಯೈ ಕೊಲುವ ಲೇದಾ (ಎಂತ)

2. ಶಿಷ್ಟುಡು ಮೌನಿ ವರಿಷ್ಟುಡು ಗೊಪ್ಪ
ವಸಿಷ್ಠುಡು ಹಿತುಡುಗಾ ಲೇದಾ (ಎಂತ)

3. ನರ ವರ ನೀಕೈ ಸುರ ಗಣಮುಲು
ವಾನರುಲೈ ಕೊಲುವಗ ಲೇದಾ (ಎಂತ)

4. ಆಗಮೋಕ್ತಮಗು ನೀ ಗುಣಮುಲು ಶ್ರೀ
ತ್ಯಾಗರಾಜು ಪಾಡಗ ಲೇದಾ (ಎಂತ)

English Lyrics

Composer: tyAgarAja
Raga: hari kambhoji
Tala: Adi
Language: telugu

Pallavi:
enta rAni tanakenta pOni nI
centa viDuva jAla shrI rAma

Anupallavi:
3 antakAri nI centa jEri
hanumantuDai koluva lEdA (enta)

Charanam:
1. shEShuDu shivuniki bhUShuDu lakShmaNa
vEShiyai koluva lEdA (enta)

2. shiShTuDu mauni variShTuDu goppa
vasiShThuDu hituDugA lEdA (enta)

3. nara vara nIkai sura gaNamulu
vAnarulai koluvaga lEdA (enta)

4. AgamOktamagu nI guNamulu shrI
tyAgarAju pADaga lEdA (enta)
