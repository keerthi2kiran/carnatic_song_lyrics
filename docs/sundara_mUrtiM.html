ಸುಂದರ ಮೂರ್ತಿಂ (sundara mUrtiM )

Kannada Lyrics

ರಚನೆ: ಮುತ್ತುಸ್ವಾಮಿ ದೀಕ್ಷಿತರ್
ರಾಗ: ಠಕ್ಕ
ತಾಳ: ರೂಪಕ
ಭಾಷೆ: ಸಂಸ್ಕೃತ

ಪಲ್ಲವಿ:
ಸುಂದರ ಮೂರ್ತಿಂ ಆಶ್ರಯಾಮಿ
ಶಿವ ಭೂ-ಸುರ ಕುಲಾಬ್ಧಿ ಚಂದ್ರಂ
ಕರುಣಾ ಸಾಂದ್ರಮ್

ಅನುಪಲ್ಲವಿ:

ಚರಣ:
ಕಂದರ್ಪ ವಿಡಂಬನ ವಿಗ್ರಹ ಕಾಂತಿಂ
ಕಾಮ ಕ್ರೋಧಾದಿ ರಹಿತೋಪಶಾಂತಿಂ
(ಮಧ್ಯಮ ಕಾಲ ಸಾಹಿತ್ಯಮ್)
ಮಂದ ಸ್ಮಿತ ಮೃದು ಗದನಂ
ಮಾಣಿಕ್ಯ ರುಚಿರ ಸದನಂ
ಕುಂದ ಮುಕುಳ ನಿಭ ರದನಂ
ಗುರು ಗುಹ ಮೋದಿತ ಗದನಮ್
