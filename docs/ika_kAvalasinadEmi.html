ಇಕ ಕಾವಲಸಿನದೇಮಿ (ika kAvalasinadEmi )

Kannada Lyrics

ರಚನೆ: ತ್ಯಾಗರಾಜ
ರಾಗ: ಬಲಹಂಸ
ತಾಳ: ಆದಿ
ಭಾಷೆ: ತೆಲುಗು

ಪಲ್ಲವಿ:
ಇಕ ಕಾವಲಸಿನದೇಮಿ ಮನಸಾ
ಸುಖಮುನನುಂಡವದೇಮಿ

ಅನುಪಲ್ಲವಿ:
ಅಖಿಲಾಂಡ ಕೋಟಿ ಬ್ರಹ್ಮಾಂಡ ನಾಥುಡು
ಅಂತರಂಗಮುನ ನೆಲಕೊನಿಯುಂಡಗ (ಇಕ)

ಚರಣ:
1. ಮುಂದಟಿ ಜನ್ಮಮುಲನು ಜೇಸಿನಯಘ
ಬೃಂದ ವಿಪಿನಮುಲಕ್-
ಆನಂದ ಕಂದುಡೈನ ಸೀತಾ ಪತಿ
ನಂದಕ ಯುತುಡೈಯುಂಡಗ (ಇಕ)

2. ಕಾಮಾದಿ ಲೋಭ ಮೋಹ ಮದ
ಸ್ತೋಮ ತಮಮ್ಮುಲಕುನು
ಸೋಮ ಸೂರ್ಯ ನೇತ್ರುಡೈನ ಶ್ರೀ
ರಾಮಚಂದ್ರುಡೇ ನೀಯಂದುಂಡಗ (ಇಕ)

3. ಕ್ಷೇಮಾದಿ ಶುಭಮುಲನು ತ್ಯಾಗರಾಜ
ಕಾಮಿತಾರ್ಥಮುಲನು
ನೇಮಮುನನಿಚ್ಚು ದಯಾ ನಿಧಿ
ರಾಮಭದ್ರುಡು ನೀಯಂದುಂಡಗ (ಇಕ)
